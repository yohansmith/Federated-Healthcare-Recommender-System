# Enhancing Scalable and Private Big Data Analytics for Personalized Healthcare

This project implements a privacy-focused predictive system designed to identify diabetes risks using **Federated Learning**. By utilizing a decentralized training approach, the system ensures that sensitive medical data remains on local devices while still contributing to a robust global model.

## 🚀 Key Features
* **Federated Learning:** Local model training and weight aggregation to maintain data privacy.
* **Class Imbalance Handling:** Implementation of **SMOTE** to improve recall for high-risk diabetic cases.
* **Explainable AI (XAI):** Integration of **SHAP** values to provide transparency into how the model makes specific predictions.
* **Interactive Dashboard:** A **Streamlit**-based frontend for real-time risk assessment and recommendations.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** PyTorch, Scikit-learn
* **Explainability:** SHAP
* **Web Framework:** Streamlit
* **Data Processing:** Pandas, NumPy, Imbalanced-learn (SMOTE)

## 📊 Results
The model successfully addressed significant class imbalances, leading to a marked improvement in the system's ability to correctly identify high-risk individuals without compromising data privacy.
